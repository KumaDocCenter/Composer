
【Composer】


WebSite: http://getcomposer.org/
GitHub:  https://github.com/composer/composer
维基百科：https://en.wikipedia.org/wiki/Composer_(software)

-----------------------------------------------------

Composer 是 PHP 的一个包依赖管理工具 ，不是一个包管理器。 
能够帮助我们安装我们项目所需要的依赖包。


(非官方)中文站点：https://www.phpcomposer.com/
配置国内镜像：https://pkg.phpcomposer.com/
